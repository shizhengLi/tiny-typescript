# TypeScript 学习文档

这是一个按学习路径组织的 TypeScript 中文专栏仓库，内容从入门概念一路推进到工程实践与类型设计。

完整目录见：

- [docs/typescript-series/README.md](./docs/typescript-series/README.md)

建议按编号顺序阅读，并把示例代码亲手敲一遍。

## 可运行示例项目

这套专栏现在配了一套可运行的 TypeScript 示例项目，代码入口在：

- [src/index.ts](./src/index.ts)
- [src/chapters](./src/chapters)
- [package.json](./package.json)

安装依赖：

```bash
npm install
```

常用命令：

```bash
npm run list
npm run chapter 01
npm run chapter 07
npm run all
npm run typecheck
npm run build
```

示例项目和文档一一对应。比如你读到泛型时，可以直接运行第 7 章示例；读到模块和声明文件时，可以直接运行第 12 章示例。
